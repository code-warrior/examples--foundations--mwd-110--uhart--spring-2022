Working with Length Units

Read more about length units at mdn

Relative Length Units

  • px — Stands for pixel, or picture element, changes in size between different devices, but is fixed according to the device.
  • em — The size of the typeface, which, for our purposes, is a default of 16 pixels.
  • rem — The size of the typeface defined by the root/html element. The default is 16 pixels.
  • ex — The size of a lowercase “x”.
  • vw — Stands for viewport width. Is 1/100th of the width of the viewport, or 1%.
  • vh — Stands for viewport height. Is 1/100th of the height of the viewport, or 1%.
  • % — Percentage of the parent element.

Fixed-Length/Absolute Units

  • in — Inches
  • cm — Centimeters
  • mm — Millimeter
  • pt — Points
  • pc — Picas
  • Q — Quarter of a millimeter
